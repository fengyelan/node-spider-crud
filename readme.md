##### 网络爬虫

* 爬取的是[图片网站](https://pixabay.com/zh/photos)

* 爬取到图片之后，在本地的pics的文件夹保存了一份爬取到的相关图片，并且把图片的相关信息保存在数据库里面

* 存储图片的文件夹：/pics/东方明珠1521084196584  ，文件夹命名方式是图片关键词+时间戳 

##### 启动命令

* nodemon app.js 
 
  ps :默认是是爬取的华山的图片相关信息

* nodemon app.js 爬取图片的名称  

  ps: 例如 nodemon app.js 故宫
